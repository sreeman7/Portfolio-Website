# Portfolio Website

A responsive personal portfolio site built with plain HTML, CSS, and JavaScript.

## Quick start

1. Open `script.js`
2. Edit the `portfolioData` object at the top:
   - `name`, `role`, `headline`, `summary`
   - `metrics`
   - `projects`
   - `skills`
   - `contacts` (`email`, `github`, `linkedin`, `resume`)
3. Open `index.html` in your browser

## Files

- `index.html`: layout and page structure
- `styles.css`: styling, layout, animations, responsive behavior
- `script.js`: all editable content and rendering logic

## Deploy options

- GitHub Pages
- Netlify
- Vercel
